# Simple Demo for Volley

    Volley 是 Google I/O 2013推出的网络请求库。Volley比较适合快速网络访问情景。


## USE Volley

```
    git clone https://android.googlesource.com/platform/frameworks/volley
    cd volley
    android update project -p
    ant jar

```
